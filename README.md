# Criar um Employee:
Ou o cnpj ou o companyName podem ser utilizados para 
indicar a empresa da qual o funcionário faz parte.

    {
        "username":,
        "password":,
        "cpf":,
        "name":,
        "cnpj":,
        "companyName":,
        "role":"employee",
    }
